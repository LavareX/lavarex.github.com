Windows 使用技巧

May 25, 2014

介于正式使用 RMBP，以后应该没多少机会使用 Windows，在此记录下之前使用 Window 的一些心得，还有纪念下陪伴我大学生活的 Acer 4752G 本本。

记得在大一暑假的某一天，冥冥之中闲逛到 X-Force 的 异次元软件世界，在 “本年热门” Tag 看到一篇文章 《让电脑更高效的魔法 - 浅谈高效率PC工作环境的简化、净化与建立方法 (上)》、《让电脑更高效的魔法 – 浅谈高效率PC工作环境的简化、净化与建立方法 (下)》，于是乎无聊没事做，就去效仿了下。

高效、Geek、神器、Auto Hot Key、Vim、Total Commander、Evernote，渐渐的开拓了我菜鸟的视野。PS：那时还不知道有 Mac 这个东东的存在

主要原则：在更短时间里，完成你要做的事

开机启动（尽量少启动项）

  • 管理开机启动项可以用系统的msconfig或者 360软件管理（介于 360安全卫士 只有这个功能有点用处，强大的网友专门提取了这个功能作为一个 .exe 文件）
  • 杀毒软件（Windows Security Essential 或者裸机）
  • Auto Hot Key 的脚本
  • 一些小的辅助应用，不影响开机速度（eg: Flux）
  • 有些大的软件真的要开机启动的话，可以用 Auto Hot Key 编写延迟启动的脚本

Auto Hot Key 的使用（尽量用快捷键启动软件）

  • 常用软件通过编写 Auto Hot Key 脚本可以快捷键启动
  • 其余的软件统一快捷方式放在同一个文件夹里，并编写脚本快捷键打开该文件夹
  • 编写一个全局开关，负责打开与关闭 Auto Hot Key 的热键设置（eg: F10::suspend,通过 F10 作为开关）

我自己的 Auto Hot Key 脚本

  • ! -> Alt按键，^ -> Ctrl按键
  • !x -> Alt + X
